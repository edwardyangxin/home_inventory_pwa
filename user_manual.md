# 使用手册

## 概览
这是一个家庭库存与习惯管理的 PWA，可通过语音或文字进行录入、查询、更新，并支持编辑和删除。每次更新后会在下方结果区展示变化内容，点击返回即可查看全量列表。

## 开始使用
- 用浏览器打开应用。
- 首次使用需允许麦克风权限（语音输入必需）。
- 左上角语言按钮可切换 `zh-CN` / `en-US`。

## 主要流程（语音或文字）
1. 点击 **开始录音** 说话，或直接在主输入框输入文字。
2. 结束录音（或说“over”）后，系统会解析内容。
3. 如为更新操作，会在短暂倒计时后自动提交。
4. 更新结果会显示在下方 **结果列表**，点 **返回** 可回到全量列表。

## 库存与习惯切换
- 在列表上方使用 **Tab** 切换 **库存** / **生活习惯**。
- 只显示一个列表，避免内容过长。

## 查询与结果展示
- 查询或更新完成后会进入 **结果模式**。
- 点击 **返回** 恢复到当前 Tab 的全量列表。

## 编辑与删除
- 点击卡片可编辑内容。
- 点击垃圾桶图标删除记录。

## 生活习惯管理
- 习惯通过主输入框语音/文字更新。
- 更新完成后会在结果列表中显示本次影响的习惯。

## 饮食推荐
- 点击厨师帽按钮，生成基于库存的 3 天游菜单建议。

## 本周采购推荐
- 点击页面顶部的 **本周采购** 按钮，生成结合库存与生活习惯的本周采购清单。
- 推荐列表会展示建议购买数量、原因、当前库存与最近一次购买时间。

## 使用提示
- 录音时说“over”可快速停止。
- 更新后请查看结果列表确认变化。
- 若数据不是最新，点击当前 Tab 的刷新按钮。
